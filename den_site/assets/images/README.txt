DEN — Image Assets Placeholder
================================

This folder is reserved for any image assets you want to add.

Currently, the DEN website uses CSS-generated graphics and SVG icons only,
so no image files are required by default.

If you wish to add custom images, place them here and reference them in
the HTML files like: <img src="assets/images/your-image.png" ...>

Suggested images to add (optional):
- og-image.png          → Open Graph / social share image (1200x630px)
- favicon.ico           → Browser tab favicon
- apple-touch-icon.png  → iOS home screen icon (180x180px)
- app-screenshot-1.png  → App screenshot for the hero or download section
- app-screenshot-2.png  → Additional screenshot

To add a favicon, place favicon.ico here and add to each HTML <head>:
<link rel="icon" href="assets/images/favicon.ico" type="image/x-icon">
